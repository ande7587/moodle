Activity Clipboard

Purpose

The Activity Clipboard is a block that enables sharing of Moodle content (resources, activities) between multiple courses on your site. You can share among teachers or among your own courses. It copies and moves single course items without user data — similar to the “Import” function in Course Administration. Items can be collected and saved on the Activity Clipboard indefinitely, serving as a library of frequently used course items available for duplication. It is viewable only by teachers, course creators and administrators. It requires Moodle version 2.0 or later and works with Topic and Weekly course formats.

Operation

Turning On and Off
With editing on, add the Activity Clipboard block from the block menu. wherever you roam courses across a site. Click the "X" icon to turn "off" the Activity Clipboard, although the contents of the Activity Clipboard will remain there when you turn it on again.
Copying to the Activity Clipboard
You will notice a small "Copy to Activity Clipboard" icon which appears after each resource or activity in the center column of a Moodle course. Click on that icon to send a copy of that resource/activity into the Activity Clipboard. User data will be removed, only the activity itself will be cloned.
Copying from the Activity Clipboard
Click "Copy to course" icon in Activity Clipboard, then the target markers appear on each section. You can select one of them to complete the copying, or "Cancel" which is above those.
Making folders inside the Activity Clipboard
Click the "Move into folder" icon on a course content item and an input box will appear if there is no folder. Input the name of the target folder there. To move an item into an existing folder, click on the "Move into folder" icon, then a drop-down list appears, and select the target folder. If you click the "Edit" icon, you will replace the drop-down list with an input box.
